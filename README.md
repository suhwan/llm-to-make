# LLM-to-Make

> 비개발자가 자연어로 Make.com 자동화 시나리오를 만들 수 있도록 지원하는 프로젝트

## 시작하기

1. 이 저장소를 클론합니다
2. `.env.example`을 `.env`로 복사하고 API 키를 설정합니다
3. `/setup` 명령어로 환경 설정을 시작합니다

## 사용 가능한 스킬

| 스킬 | 용도 |
|------|------|
| `/setup` | 온보딩, 환경 설정 |
| `/new-scenario` | 시나리오 생성 |
| `/check-tables` | 테이블 구조 확인 |
| `/status` | 프로젝트 현황 |
| `/analyze` | 시나리오 분석 |
| `/backup` | 청사진 백업 |
