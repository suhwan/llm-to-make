# LLM-to-Make 프로젝트 규칙

> **목적**: 비개발자가 자연어로 Make 시나리오를 만들 수 있도록 지원

## 첫 실행 안내

1. `/setup` 명령어를 실행하세요
2. `.env` 파일을 설정하세요
3. 상세 가이드: [docs/onboarding.md](docs/onboarding.md)

## 사용 가능한 스킬

| 스킬 | 용도 |
|------|------|
| `/setup` | 온보딩, 환경 설정 |
| `/new-scenario` | 시나리오 생성 |
| `/check-tables` | 테이블 구조 확인 |
| `/status` | 프로젝트 현황 |
| `/analyze` | 시나리오 분석 |
| `/backup` | 청사진 백업 |
